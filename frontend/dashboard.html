How Settings Are Used

              
Saved profile and prompt settings are automatically applied in both agents.

              

                

                  Content Brief Agent
                  
Uses brand profile, selected brief personality, custom brief personality notes, and custom brief prompt override while generating the editable content brief.

                

                

                  Content Writing Agent
                  
Uses brand profile, selected writer personality, custom writer personality notes, and custom writer prompt override while generating article drafts from a brief.

                

                

                  AI Workspace
                  
Uses the selected AI Workspace personality and custom workspace personality notes while deciding whether to create a brief, write content directly, or route through a saved brief.
